\# Poverty and social exclusion in Spain (2014–2025)



Exploratory analysis of AROPE trends in Spain using official data from the Spanish National Statistics Institute (INE).



\## Contents

\- Exploratory article in PDF format

\- R analysis script

\- Dataset used for the analysis



\## Tools

\- R

\- tidyverse

\- ggplot2

\- gt (table generation)



\## Main focus

The project explores:

\- gender inequalities

\- age-based vulnerability

\- evolution of poverty and social exclusion risk

\- structural patterns in AROPE trends



This project was independently developed as part of a broader transition towards applied quantitative political and social analysis using R.
